# Water-saving-module
This is the data gatherer for a bigger project that will collect ambient data to save water while watering plants.
Hello, my name is Alexis. I am part of ACOIIN in Chile, and I designed this PCB to save water during the process of watering plants.

Most people don’t know how much water plants actually need to be watered efficiently, so I designed this small ATtiny85-based data gatherer to interface with other modules that will help water plants more efficiently—saving water, saving money, and helping the environment.
